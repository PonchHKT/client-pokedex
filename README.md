## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

> client-pokedex/client
> client-pokedex/server
```sh
npm install
```

## Usage

> client-pokedex/client
```sh
npm start
```
> client-pokedex/server/src
```sh
node server.js
```

## Run tests

> client-pokedex/client
```sh
npm test
```
